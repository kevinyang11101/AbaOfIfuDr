# 宠物管理系统 java975

## 项目概述

宠物管理系统是一款基于Java前后端分离的Spring Boot框架开发的应用，旨在为宠物行业提供全面的管理解决方案。

## 技术栈

- **开发语言**：Java
- **后端框架**：Spring Boot、MyBatis
- **前端框架**：Vue3
- **数据库**：MySQL
- **数据库管理工具**：Navicat
- **JDK版本**：JDK8
- **构建工具**：Maven（Maven3+）
- **推荐浏览器**：谷歌浏览器
- **架构模式**：前后端分离

## 系统角色

系统设定了以下三种角色：

- 管理员
- 医生
- 用户

## 功能模块

### 管理员

- **登录**
- **个人中心**
  - 个人信息
  - 修改密码
- **用户管理**
- **医生管理**
- **商品信息管理**
- **商品评价**
- **流浪动物管理**
- **宠物资讯管理**
- **在线交流管理**
- **轮播图管理**

### 医生

- **登录**
- **个人中心**
  - 个人信息
  - 修改密码
- **预约信息管理**
- **病例档案管理**
- **在线交流管理**
- **退出**

### 用户

- **注册登录**
- **个人中心**
  - 个人信息
  - 修改密码
- **首页**
- **医生信息**
- **医生预约**
- **交流论坛**
- **商品信息**
- **在线交流**
- **宠物资讯**
- **流浪动物信息**
- **退出**
